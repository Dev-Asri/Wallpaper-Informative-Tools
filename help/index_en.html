1. Introduction

WallpaperInformativeTool is a program that displays system information (CPU, RAM, disks, network, OS, database, etc.) on your Windows wallpaper. It works similarly to Microsoft's BGInfo tool but offers a more modern interface and greater customization.

  • Completely free and open source
  • Bilingual interface (Türkçe / English)
  • Drag & drop item positioning
  • Auto-alignment (left, right, top, bottom)
  • Per-item font, color, and size settings
  • Multi-select for bulk moving and alignment

2. Installation & Running

EXE Version (Recommended)

  1. Copy WallpaperInformativeTool.exe to any folder
  2. Double-click to run — no installation required, fully portable

Running with Python (For Developers)

pip install -r requirements.txt
python src\main.py

Requirements: Python 3.10+, Pillow, psutil, wmi, pyodbc (optional), pyinstaller (optional).

Building the EXE

pyinstaller --onefile --noconsole --name "WallpaperInformativeTool" ^
  --add-data "windows_sysinfo;windows_sysinfo" ^
  --hidden-import wmi --hidden-import getpass --hidden-import pyodbc src\main.py

The program consists of three main sections:

4. Information Sources

Four checkboxes on the toolbar determine which data sources to collect from:

SourceCollected Information
NetworkHostname, IP addresses (IPv4/IPv6), network interface info
HardwareCPU (name, cores, speed, usage), RAM (total, used, free), disks (size, usage), GPU, motherboard
SystemOS (version, build, architecture), user info, uptime, timezone, license, activation, user list
DatabaseSQL Server instances, ODBC connections (system/user DSN)
Only select the sources you need — this reduces collection time and clutter.

5. Information Tree (Available Information)

The tree on the left panel shows all info items from selected sources, organized by category. Each item displays its current value (or the cached value).

  • To add an item: Double-click an item → it's added to the Active Items list
  • Preview: Added items appear in the preview area on the right
  • Cache: Data is collected via the "Refresh Data" button; cached data is shown in the tree

6. Add, Remove, and Reorder Items

Adding

  • Double-click an item in the left tree
  • The item is added to the Active Items list and the preview
An error is shown if you try to add an already active item.

Removing

  • Select a row in the Active Items list
  • Click ❌ Remove
  • Or click 🔄 Clear to remove all active items

Reordering

  • Use ▲ Move Up / ▼ Move Down buttons
  • The order determines the display order on the wallpaper

7. Dragging Items

You can drag items in the preview area to position them anywhere.

Basic Dragging

  1. In the preview area, click on an item's label or value
  2. Hold the mouse button and drag
  3. Release to place the item at the new position

Drag Constraint Modes

Use the "Drag Mode" section on the right panel to constrain movement:

ModeBehavior
↔ Horizontal (only)Items move only on the X axis
↕ Vertical (only)Items move only on the Y axis
Both selected / Neither selectedItems move freely in any direction

Multi-Select Bulk Move

  1. Hold Ctrl and click on multiple items in the preview
  2. Selected items are highlighted
  3. Drag any selected item → all selected items move together
Ctrl+Click can select only the label or only the value of an item. This lets you move labels and values independently.

Panning the Preview

Click and drag on an empty area of the preview to pan the entire view. This clears the current selection.

8. Alignment

Four alignment buttons arrange selected items relative to each other:

ButtonAction
← Align LeftSets the X coordinate of all selected parts to the leftmost item's X
→ Align RightSets the right edge (X + width) of all selected parts to the rightmost item's edge
↑ Align TopSets the Y coordinate of all selected parts to the topmost item's Y
↓ Align BottomSets the bottom edge (Y + height) of all selected parts to the bottommost item's edge
Important: Alignment measures the actual pixel width of text (using Pillow). Right and bottom alignment correctly account for each part's width/height.

Auto-Align (Align Button)

The 📋 Align button arranges all active items at default positions (origin: 30,30; label-value gap: 200px; row height: 35px). Useful for quickly organizing scattered items.

Alignment Tips

  • First select at least 2 items with Ctrl+Click
  • You can select only labels or only values for separate alignment
  • Align labels of different items together, values in a separate group

9. Properties Panel

Below the center panel, you can edit all properties of the selected item. It has three tabs:

Label Tab

  • Label Name — Change the displayed text
  • Static Value — Show a fixed text instead of the dynamic value
  • Show — Toggle label visibility
  • Font / Size / Color — Customize appearance

Value Tab

  • Show — Toggle value visibility
  • Font / Size / Color — Customize appearance
  • Static Value — Show fixed text as the value

Position Tab

  • Label X / Y — Pixel position of the label
  • Value X / Y — Pixel position of the value
  • Enter values directly for precise positioning

Apply to All

The → All button next to each property applies that property to all active items at once. For example, change the font of all labels simultaneously.

10. Row Spacing

The "Row Spacing" spinbox at the bottom of the right panel rearranges the vertical (Y) positions of all active items. When you change the value:

  • Items' Y coordinates are evenly distributed starting from the first item's Y
  • X coordinates are preserved
  • Range: 10-100 pixels

11. Background Settings

Two background buttons on the right side of the toolbar:

Background Color

Click 🎨 Background Color to choose a solid background color. Default: dark navy (#1e1e2e).

Background Image

Click 🖼 Background Image to select an image file (JPG, PNG, BMP). The image is scaled to fit the screen resolution.

The background type (color or image) is determined automatically. The last selected type is used.

12. Language Switching (Türkçe / English)

The toolbar has Türkçe / English radio buttons. Click to switch the entire interface instantly:

  • Menus, buttons, labels
  • Tree item names and categories
  • Status bar messages
  • Dialog windows (warning, success, confirm)

Your language preference is saved and remembered on next launch.

When you switch languages, your active item labels are updated while keeping your position settings.

13. Startup Settings

Refresh on Startup

If checked, data is automatically collected and cached when the program starts. Items show their values immediately if cache is available.

Change & Close on Startup

This feature is for running the program at Windows startup to automatically change the wallpaper and exit:

  1. Check the box
  2. Set the wait duration (1-10 seconds)
  3. On next launch:
    • Starts minimized to taskbar
    • Collects data
    • Applies the wallpaper
    • Waits the set duration
    • Closes automatically
Cancellation: Before the duration expires, open the program window and uncheck "Change & Close on Startup" — the program will not close and continues normally.

14. Applying the Wallpaper

  1. Make sure you have at least one active item
  2. Position items and adjust properties as desired
  3. Choose a background color or image
  4. Click 🖨 Apply Wallpaper

The program:

  1. Detects the physical screen resolution (DPI-independent, real pixels)
  2. Creates a high-quality BMP image
  3. Renders the information text onto the image
  4. Changes the wallpaper via the Windows API
Resolution: The program ignores Windows DPI scaling. It creates a lossless image at the physical screen resolution (100% scale), ensuring sharp, readable text.

15. Preview

The preview area on the right panel shows how the wallpaper will look.

Live Preview

As you add items, the preview updates in real time. All changes (position, font, color) are reflected immediately.

Full Screen Preview (🔍 Preview)

  • Displays the wallpaper in full screen
  • Mouse wheel to zoom in/out
  • Press ESC or click to close

16. Desktop Preview

🖥 Desktop Preview shows the wallpaper on your actual desktop:

  1. All open windows are minimized
  2. The wallpaper is temporarily applied
  3. It's displayed for the set Wait Duration (1-60 seconds)
  4. After the time expires, the old wallpaper is restored and the app comes to the front
Desktop preview is ideal for testing the wallpaper in the real environment. Adjust the wait duration to your needs.

18. Frequently Asked Questions

Why won't the program open?

EXE version: antivirus software may be blocking it. Python version: make sure you installed dependencies with pip install -r requirements.txt.

Why are values not showing?

Click the "Refresh Data" button. On first launch, items may show "..." until data is collected.

Why won't the wallpaper apply?

Make sure you have at least one active item. The wallpaper cannot be applied with an empty Active Items list.

Where are settings stored?

In %APPDATA%\WallpaperInformativeTool\settings.json. Delete this file to reset all settings.

Where is the cache stored?

cached_data.json is stored next to the EXE or in the project folder. It's updated when you refresh data.

How do I change the language?

Click the Türkçe / English radio button on the toolbar. The interface changes instantly.

Does it work with multiple monitors?

The program uses the primary monitor's physical resolution. Multi-monitor support is limited.

Can I run it at Windows startup?

Add a shortcut to the EXE in shell:startup and check "Change & Close on Startup".
